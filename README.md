## Getting started

Set up the app with these commands

```
cd frontend
yarn install
cd ..
cp .env.example .env

```

Set the port in the new envfile, then run:

```
docker-compose up
```

after suffessfull setup go to `http://localhost:<YOUR_PORT>`
